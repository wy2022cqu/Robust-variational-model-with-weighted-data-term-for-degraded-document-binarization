# Robust-variational-model-with-weighted-data-term-for-degraded-document-binarization

This repository contains the MATLAB implementation of our paper:  
**“Robust variational model with weighted data term for degraded document binarization.”**

The repository includes test data and can be run directly.

## Files and folders

- `Binary_demo.m`: main demo script. Run this file directly to obtain the binarization result.
- `gt`: contains the ground-truth segmentation result for the image `110_20284.bmp`, together with the two weight files used for metric computation.
- `test_one`: stores the binarization result generated by the code.

## Usage

Run `Binary_demo.m` directly in MATLAB to obtain the binarization result.

## Note

To use `DIBCO_metrics.exe` to compute the four evaluation metrics (**FM, Fps, PSNR, and DRD**), the MATLAB Runtime must be installed in advance.
